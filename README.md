# Welcome 
Welcome to the Scroll-Bar - ver 1.1.0

# Setup
Import style and scripts for web page

```html
    <!-- Style -->
    <link rel="stylesheet" href="css/mofi-scroll-bar.css" />

    <!-- Scripts -->
    <script src="js/scroll_bar.mini.js"></script>
```

# Initialization
```javaScript
    <script>
        window.onload = mofiScrollBar('#slider');
    </script>
```
